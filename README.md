# User_Login -- Authentication System

## Overview
A standalone, single-file authentication interface with Login, Signup, and Password Reset functionality.

## Files
- **`index.html`** — Complete standalone authentication page (HTML + CSS + JS in one file)

## Features
- **Login** — Email/Username + Password with visibility toggle
- **Signup** — Full Name, Username (@prefix), Email, Password, Confirm Password
- **Forgot Password** — Email-based reset link
- **Google Sign-In** — OAuth button (wire to your backend)
- **Responsive** — Mobile-friendly layout
- **Clean Design** — White background, Inter + DM Sans fonts, blue accent

## Usage
Open `index.html` directly in any browser — no build tools or dependencies required.

## Customization
- Edit CSS variables in `:root` to change colors, fonts, and spacing
- Wire form submissions in the `<script>` section to your backend API
- Replace the logo by updating the `<img>` src in the header

## Tech Stack
- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Inter, DM Sans)
- No frameworks or dependencies
